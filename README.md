# CWT 命題工作平臺 (官方版)

## 網站介紹

「CWT 命題工作平臺」官方版是一個基於 Blazor Server 技術開發的進階線上命題與審核管理系統。此平台旨在提供一個高效、安全且功能全面的環境，支援全民中文檢定（CWT）的試題從命製、多階段審核、修訂到最終入庫的完整生命週期管理。系統針對不同使用者角色（如系統管理員、命題教師、審核委員等）提供精細的權限控制，並整合了即時通訊、數據分析、相似度比對等多項功能，以確保試題品質、提升工作效率並優化使用者體驗。

## 主要功能

*   **高安全性使用者認證與權限管理**：基於 ASP.NET Core Cookie Authentication，提供安全的登入機制，並透過 `IRoleService` 和 `IMembershipService` 實現精細的角色與模組權限控制。
*   **多階段試題審核流程**：支援「命題」、「互審」、「專審」、「總審」等多個審核階段，並由 `IQuestionService`、`IReviewService` 及 `IPhaseTransitionCoordinator` 協調各階段流轉，確保試題品質與嚴謹性。
*   **即時試題狀態追蹤與通知**：透過 SignalR 實現即時通訊，確保各角色能即時獲取試題狀態更新、公告與急件提醒，提升協作效率。
*   **靈活的試題內容管理**：支援多種題型（如單選題、閱讀題組、聽力測驗等），提供豐富的編輯器（Quill Editor）與預覽功能，確保試題內容的完整性與呈現效果。
*   **數據概覽與報表匯出**：`IDashboardService` 提供關鍵數據概覽，並支援將試題報表匯出為 Excel 格式（透過 NPOI 函式庫），方便管理者掌握整體命題進度與成果。
*   **相似度分析**：整合 `ISimilarityService` 進行試題內容的相似度比對，協助審核委員判斷試題原創性。
*   **審後修訂機制**：提供 `IRevisionService` 支援管理員在三審結束後對試題進行再編輯與決策調整。
*   **聘書自動繪製與下載**：`IAppointmentService` 支援自動生成聘書並打包下載，簡化行政流程。

## 技術棧

*   **後端框架**：ASP.NET Core 10 (Blazor Server)
*   **程式語言**：C#
*   **資料庫存取**：Dapper (輕量級 ORM), Microsoft.Data.SqlClient
*   **即時通訊**：ASP.NET Core SignalR
*   **前端技術**：Blazor Components, HTML5, CSS3 (Tailwind CSS), JavaScript (部分客戶端互動)
*   **其他函式庫**：HtmlSanitizer (HTML 清理), NPOI (Excel 檔案操作)
*   **開發工具**：Visual Studio (推測)
